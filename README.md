# PLUGIN-DEMO
说明： 这是我写的插件总的一个DEMO。

使用技术栈：  webpack + es6 + vue + vue-router

查看方式

  第一步  git clone git@github.com:Mrangmaomao/PLUGIN-DEMO.git
  
  第二步  npm install 安装依赖
  
  第三步  npm run dev
  
  整个项目启动成功， 在浏览器输入： http://localhost:9060 就可查看
